CATTSS

Computer Assisted Transcription Through Subword Spotting

This is the crowd-source transcription app I've created as part of a thesis at BYU. It is prototyping a new paradigm for historical document transcription.

It is intended for a touch device, though it should work in any modern browser.

I'm running a short experiment which should take about 10 minutes.

I'll help!

How CATTSS works

The system begins by looking for common subwords ('th', 'er', etc.) in the text. It can't do this very well by itself, so it passes subwords it is unsure about on to users. With this feedback the system eventually reaching a point when it can make a decision about the rest of them.

Together, subwords form a partial transcription of words (think of the game "Wheel of Fortune", some letters are known, some are blank). Comparing these to a lexicon, the system determines the possible transcriptions for a partially transcribed word. If this list is reasonably small the system passes the list to a user to select the correct one.

The system also uses the subwords approved by users, and the now known letters of user transcribed words to enhance its ability to find subwords.
